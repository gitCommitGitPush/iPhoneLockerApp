# iPhoneLockerApp
## First Project - Enter Code 404 to unlock the locker.
## Code Tips
1. Add sound by Adding New Lib. Gen > Link Framework
2. Use 'let' to create constant
3. Use 'Void' for non-returning functions
4. Use '\( var1 )' to print variable
5. Use 'inout' in function declaration to pass by reference
6. Use '&' on function call to pass by reference
7. Use \<var> : \<datatype> in function declarations
8. Use viewDidLoad in controller to set initial state
